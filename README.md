# AR Signature Hospitality

Professional corporate catering and luxury hampers project.

## Project Structure
- **/src**: Angular 17 Frontend (Standalone Components)
- **/backend**: Spring Boot Backend
- **angular.json**: Build configuration
- **package.json**: Frontend dependencies

## Deployment
- **Frontend**: Deployed to Vercel (Auto-detected from root)
- **Backend**: Deploy to Render / Railway / Cloud Run
